> A number of small projects for the System Development Board based STM32F103C8T6

## Dependensies

- CMSIS
- STM32F1xx_HAL_Driver
- STM32CubeMX 5.4.0

## Projects

> HAL Projects created using STM32CubeMX. CMSIS Projects created using exclusively CMSIS.

### LED blinking

> Hello world project. Makes the LED blink once a second. Interrupt timer 4 is used. The LED is connected to the PC13.
